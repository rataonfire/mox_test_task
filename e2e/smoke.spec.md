# MOX Smoke Test — Ферма невидимых кроликов

Приёмочный тест: человек или Playwright-агент выполняет шаги против http://localhost:4173 (`npm run preview`), http://localhost:5173 (`npm run dev`) или прод-URL.

---

## Предварительные условия

- [ ] `npm install && npm run build && npm run preview` (или `npm run dev` для локальной разработки)
- [ ] Браузер открыт на адресе выбранного сервера (см. выше)
- [ ] DevTools Console открыта; на загрузке нет ошибок (красных записей)

---

## Тест 1: Загрузка и исходные данные

- [ ] Hero-карта показывает **≈ 9 кроликов (от 7 до 11)**
- [ ] Шкала уверенности показывает **62% (средняя)** с жёлтым цветом
- [ ] Секция «Что повлияло на оценку» содержит 5 строк-вкладов (top 5)
- [ ] Верхние 3 строки выделены (жирный текст)
- [ ] Секция «По локациям» показывает 4 карточки: Огород, У забора, Сарай, Теплица
- [ ] Секция «Рекомендации» показывает ≥2 карточек (инфо, варнинг, алерт)
- [ ] Таблица сигналов показывает 5 строк (evt_001 … evt_005), все включены (чекбоксы ☑)
- [ ] Кнопка «Параметры модели» видна; панель свёрнута
- [ ] Во вкладке **«Данные»** (текущая вкладка видна)
- [ ] Во вкладке **«AI Worklog»** доступна (кликабельна)
- [ ] Кнопка **«Данные JSON»** видна в шапке
- [ ] Кнопка **«Сбросить к исходным данным»** видна в шапке

## Тест 3: Параметры модели (слайдеры)

- [ ] Нажать **«Параметры модели»** → панель раскрывается
- [ ] Видны 8 слайдеров (или их эквивалент):
  1. Вес «Пропавшая морковь» (default 0.4, range 0.1–2.0)
  2. Вес «Новая ямка» (default 1.2)
  3. Вес «Датчик движения» (default 1.0)
  4. Вес «Шорох» (default 0.5)
  5. Вес «Следы» (default 0.3)
  6. Скидка за пересечение (default 0.6, range 0.3–1.0)
  7. Ширина неопределённости (default 0.5, range 0.1–1.0)
  8. Порог ненадёжности (default 40%, range 10–60%)
- [ ] Переместить слайдер **«Вес Новая ямка»** с 1.2 на 2.0 → оценка **увеличилась**
- [ ] Переместить слайдер **«Скидка за пересечение»** с 0.6 на 0.3 (больше скидки) → оценка **уменьшилась**
- [ ] Нажать **«Сбросить к умолчаниям»** → все слайдеры вернулись к исходным, оценка = 9
- [ ] *Примечание:* на исходных данных скидка за пересечение может не изменить округлённое число (8.94 → 9 устойчиво); используйте слайдер веса для наглядной проверки.

## Примечания для исполнителя

1. **Хронометраж:** полный прогон (все 11 тестов) занимает примерно 5–10 минут руками.
2. **Автоматизация:** тесты 1–10 легко автоматизируются Playwright; тест 11 (AI Worklog) требует проверки наличия 7 карточек и текстового содержимого.
3. **Новые локации:** при добавлении сигнала с новой локацией (тест 7) это должно автоматически создать карточку в секции «По локациям».
4. **Жаргон:** ошибки импорта обязательно на **русском языке** (например, «Не удалось прочитать JSON…», а не «Failed to parse»).
5. **Режим редактирования:** таблица сигналов может быть inline-редактируемой или открывать форму в модали — оба подхода валидны.
6. **Сброс и reload:** localStorage сохраняет состояние; при hard-refresh (Ctrl+Shift+R) данные должны восстановиться; при очистке localStorage — должны восстановиться исходные (seed) данные.
